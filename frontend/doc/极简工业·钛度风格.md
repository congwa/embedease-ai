# 「极简工业·钛度」 (Industrial Titanium) UI 风格指南

> "这不是一个聊天窗口，而是一次**深思熟虑的校准**。每一个像素都像是在 40 吨的压力机下冲压而成，坚固、沉默、精准。我们去掉了所有的浮夸，只保留冰冷的金属与温润的思维在碰撞。"

## 1. 核心视觉理念 (Core Philosophy)

*   **精密 (Precision):** 追求 0.5px 的线条精度。界面元素之间有严格的网格对齐。
*   **冷淡 (Cold):** 放弃情感化的色彩，拥抱单色调。只在关键状态（运行中、错误、成功）使用高饱和度的指示灯色。
*   **触感 (Tactile):** 摒弃“悬浮”和“投影”，改用 **“压印 (Deboss)”**、**“蚀刻 (Etch)”** 和 **“拉丝 (Brushed)”** 效果。让界面看起来具有物理厚度和阻尼感。
*   **机械反馈 (Mechanical):** 交互应当干脆利落，没有多余的缓动，只有“开”与“关”的明确状态。

---

## 2. 设计变量系统 (Design Tokens)

我们需要在全局 CSS (`globals.css`) 或 Tailwind 配置中定义以下变量。

### 2.1 色板 (Color Palette)

```css
:root[data-theme="industrial"] {
  /* --- 基底色 (Backgrounds) --- */
  /* 核心底色：氧化黑钛 */
  --bg-metal-base: #121213; 
  /* 面板色：冷加工铝 (略浅) */
  --bg-metal-panel: #1C1C1E; 
  /* 凹槽色：用于输入框或深层容器 */
  --bg-metal-inset: #0A0A0A;
  
  /* --- 文字色 (Typography) --- */
  /* 主文字：喷砂铝色 (非纯白) */
  --text-primary: #D1D1D1; 
  /* 次要文字：蚀刻灰 */
  --text-secondary: #636366;
  /* 辅助文字/元数据：暗淡灰 */
  --text-tertiary: #48484A;
  
  /* --- 装饰线与边框 (Borders) --- */
  /* 边缘高光：模拟金属倒角的反光 */
  --border-highlight: rgba(255, 255, 255, 0.1);
  /* 边缘阴影：模拟厚度 */
  --border-shadow: rgba(0, 0, 0, 0.8);
  /* 基础边框色 */
  --border-base: #2C2C2E;

  /* --- 指示灯/功能色 (Indicators) --- */
  /* 唯一的强调色：国际安全橙 */
  --color-accent: #FF4500; 
  /* 运行中：磷光绿 */
  --color-success: #00FF9D; 
  /* 警报：工业红 */
  --color-error: #FF3B30;   
  /* 待机：钨丝黄 */
  --color-warning: #FFD60A; 

  /* --- 阴影系统 (Shadows) - 核心差异点 --- */
  /* 内凹阴影：用于输入框、按下的按钮 */
  --shadow-inset-deep: inset 2px 2px 5px rgba(0,0,0,0.8), inset -1px -1px 1px rgba(255,255,255,0.05);
  --shadow-inset-shallow: inset 1px 1px 2px rgba(0,0,0,0.5);
  /* 凸起阴影：用于面板边缘 */
  --shadow-plate: 1px 1px 0px rgba(0,0,0,1), -1px -1px 0px rgba(255,255,255,0.05);
  
  /* --- 纹理 --- */
  --texture-grain: url('/assets/textures/noise-heavy.png');
}
```

### 2.2 排版 (Typography)

*   **Font Family:** 严格的工业感。
    *   代码/数据: *JetBrains Mono, Fira Code, Roboto Mono*
    *   界面: *Inter, SF Pro Display, Helvetica Neue* (紧凑字间距)
*   **Style:** 全大写 (Uppercase) 用于标签和标题。
*   **Size:** 字号偏小 (12px-14px)，强调信息密度。

## 3. 组件风格规范 (Component Specifications)

### 3.1 布局容器 (Layout)

*   **全局背景:** 
    *   `--bg-metal-base`。
    *   覆盖一层高强度的噪点 (`opacity: 0.05`)，模拟金属磨砂表面。
*   **分割线:** 所有的分割线都是“双线”的：一条深黑线 (`#000`) 紧挨着一条浅灰线 (`rgba(255,255,255,0.05)`)，形成物理刻痕感。

### 3.2 侧边栏 (ChatSidebar)

对应的组件结构：`Sidebar`

*   **容器样式:**
    *   背景 `--bg-metal-panel`。
    *   右侧边框：`1px solid #000` + 右侧 `1px solid #333` (立体分割)。
*   **列表项 (`SidebarMenuButton`):**
    *   **形状:** 直角矩形，或者极小的圆角 (2px)。
    *   **默认:** 边框透明。
    *   **Hover:** 背景 `--bg-metal-base`，文字变亮，左侧出现 `2px` 宽的 `--color-accent` 高亮条。
    *   **Active:** 背景 `--bg-metal-inset`，产生 `--shadow-inset-shallow` 效果，如同按钮被物理锁定按下。
    *   **分割线:** 每个 Item 之间可以有一条极细的刻度线。

### 3.3 顶部导航栏 (Header)

对应的组件结构：`header`

*   **样式:**
    *   高度压缩 (48px)。
    *   背景 `--bg-metal-panel`。
    *   底部边框：双线刻痕。
    *   **装饰:** 右上角增加几个状态指示灯（小圆点），模拟设备连接状态（Online/Offline）。

### 3.4 消息区域 (ChatContent)

*   **布局:** 类似终端 (Terminal) 或 IDE 日志窗口。
*   **用户消息:**
    *   **气泡:** 无气泡，或仅有左侧边框。
    *   **背景:** 透明 或 `--bg-metal-inset`。
    *   **标识:** 前缀 `USER >` 或图标。文字颜色 `--color-accent`。
    *   **字体:** 等宽字体。
*   **AI 消息:**
    *   **气泡:** 无气泡。
    *   **背景:** 透明。
    *   **标识:** 前缀 `SYSTEM :: RESPONSE >`。
    *   **文字:** `--text-primary`。Markdown 代码块使用深色高对比度主题 (e.g., `Monokai`, `Dracula`)。
    *   **光标:** 块状光标 (`█`) 闪烁，而非竖线。

### 3.5 输入区域 (PromptInput)

*   **核心概念:** "控制台操作面板 (Control Panel)"。
*   **容器:**
    *   背景 `--bg-metal-panel`。
    *   顶部有一条明显的金属分割线。
*   **输入框:**
    *   **样式:** “槽位”设计。
    *   **背景:** `--bg-metal-inset`。
    *   **阴影:** `--shadow-inset-deep` (深邃的内凹)。
    *   **边框:** `1px solid #333`。
    *   **圆角:** `2px` 或 `0px`。
*   **发送按钮:**
    *   **样式:** 实体物理按钮。
    *   **形状:** 正方形或微圆角。
    *   **背景:** 渐变灰（凸起感）。
    *   **阴影:** `--shadow-plate`。
    *   **Active:** 变为内凹，颜色变为 `--color-accent`，阴影反转为 `--shadow-inset-shallow`。

---

## 4. 动效细节 (Motion Guidelines)

*   **无拖泥带水:** 所有的 `transition-duration` 不超过 `0.15s`。
*   **故障艺术 (Glitch):** 在 AI 开始生成或报错时，可以加入微弱的文字抖动或色彩分离 (RGB Split) 效果。
*   **逐行扫描:** 文字出现时，模拟 CRT 显示器的逐行扫描感。

    }

}
